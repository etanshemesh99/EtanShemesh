Leash spooler

Shwon below is the machine I created and desinged over the summer of 2020. This machine uses removable rods, a stepper motor, arduino, and a backr-roller for support to coil blue plastic wire around the removable rods, which are then heated to create leashes. After being heated, carabiners are attached to the leashes with help from oval cable crimps for fishing uses. Tocreate this machine, many parts had to be manfactured in China, so I made professional SOLIDWORKS drawings for the machinists. I also had to write the code for the arduino, find teh right driver and stepper motor, as well as wire multiple buttons for different length leashes. These leashes are now being sold in fishing stores all across New Jersey to retail customers.

Jet Fuselage

This make-shft jet fuselage I designed is for passengers in a flight simulator, where passengers can side in this round structure in traditional jet-style seats for a hyper-realistic flight simulator. The main structure is built using quad metal extrusions with custom brackets I designed to allow for the extrusions to connect seemlessly hidden from the passengers sight. The outside has a 3003 aluminum shell and on the inside is plastic covered in leather.

Swimming Lure

This is a fishing lure I have designed for REEL INNOVATION fishing company. The displayed lure is a part used for trolling fishing which carries behind it fake squish which the fish bite on. Shown in the GIF is how the worm gear system works by using a fin powered worm shaft to cause an internal mechanism to oscillate the rudder in the water. Low density plastic floats are mounted on both ends to keep the lure afloat and allow the water to spin the worm shaft.

Egg Drop Project

This egg-catching device is a project I am proud to put my name on, as it is a project I worked very hard on by myself, and the results of my hard work showed in the testing of the design. The task assigned was to build a device that could catch an egg dropped from high heights and keep the egg in tact when it fell into the device. My device combined many different materials to maximze the time between its initial impact with the catcher, and the point at which it had 0 velocity. With assistance of nonnewtownian fluids, packing-peanuts, and a peice of a mattress pad, my device had perfomed better than any deivce that was turned in to my teacher for as long as she was doing this project. My device was caught an egg in-tact from a higher distance than anyone else who had completed this project with my teacher. Although this is not as complex and impressive as some of my other projects, this invention is where my interest in engineering peaked, and the class that made me realize the career path I want to take.
